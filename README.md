# Todo List

A simple web application to track daily tasks, built with HTML, CSS, and JavaScript.
Based on a tutorial, extended and modified independently to add new features and improve functionality.

## Technologies
- HTML
- CSS
- JavaScript

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Dynamic updates of task list

## How to Run
1. Clone the repository
2. Open `index.html` in a web browser
3. Start adding your tasks!

##  What I Learned
- DOM manipulation using JavaScript
- Handling user events and interactions
- Updating the UI dynamically
- Structuring a small web project
